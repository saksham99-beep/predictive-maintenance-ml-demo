**Predictive Maintenance ML Demo**

A Python project demonstrating a *predictive maintenance workflow* using synthetic sensor data.  
This project is for *educational purposes and portfolio showcase*.

**Overview**

This project simulates sensor readings from industrial equipment (temperature, vibration, and pressure) and uses **Logistic Regression** to predict potential equipment failures.

- **Data Simulation:** Synthetic sensor readings  
- **EDA:** Visualize data using pairplots  
- **Feature Scaling:** StandardScaler for normalized inputs  
- **Machine Learning:** Logistic Regression  
- **Evaluation:** Confusion matrix and classification report  
- **Live Prediction Example:** Predict failure from new sensor readings  

## Tech Stack

Python 3.x, NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn
